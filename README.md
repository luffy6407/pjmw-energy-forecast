# PJMW Hourly Energy Consumption Forecasting

## Objective
Forecast hourly electricity demand using machine learning and time series techniques.

## Models Implemented
- Moving Average
- Linear Regression
- XGBoost
- Holt-Winters
- LSTM

## Best Model
XGBoost

Performance:
- MAE: 209.60
- RMSE: 276.88
- R²: 0.9225

## Features Used
- Hour
- Day of Week
- Month
- Quarter
- Year
- Day of Year
- Lag24
- Lag48
- Lag168
- Rolling24 Mean
- Rolling168 Mean
- Rolling30 Day Mean

## Deployment
Streamlit Web Application
